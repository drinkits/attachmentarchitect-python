---

QUICK START WORKFLOW:

1. RUN SETUP
   Windows: setup.bat
   Linux/Mac: chmod +x setup.sh && ./setup.sh
   
   This will:
   - Create virtual environment
   - Install Python dependencies
   - Create .env from template
   - Create necessary folders

2. CONFIGURE
   Edit .env file:
   - JIRA_BASE_URL=https://your-jira.com
   - JIRA_TOKEN=your_token_here

3. TEST CONNECTION
   python test_connection.py
   
   Expected output:
   ✓ Connected to Jira successfully!

4. RUN SCAN
   python jira_dc_scanner.py
   
   This will:
   - Scan all attachments
   - Detect duplicates
   - Calculate storage metrics
   - Automatically generate HTML report

5. VIEW REPORT
   Open: reports/scan_XXXXXXXX_visual_analysis.html
   
   Report includes:
   - 6 interactive analysis tabs
   - Sortable tables
   - Search and filter
   - Visual charts
   - Actionable insights

---

ADVANCED USAGE:

Resume interrupted scan:
  python jira_dc_scanner.py --resume SCAN_ID

Custom report criteria:
  python generate_visual_analysis_report.py scan_XXXXXXXX.json --min-size 5 --min-days 180

Custom JQL query:
  Edit config.yaml → filters → custom_jql
