GETTING STARTED
================================================================================

1. Read START_HERE.txt for quick setup instructions
2. Run setup.bat (Windows) or ./setup.sh (Linux/Mac)
3. Copy .env.example to .env and add your Jira credentials
4. Run test_connection.bat/sh to verify connection
5. Run run_scan.bat/sh to start scanning

For detailed information, see README.md
