LAUNCHER SCRIPTS - No Manual Virtual Environment Activation!

Windows Users:

  - run_scan.bat → Automatically activates venv and runs scanner

BENEFITS:
  ✓ No need to remember "venv\Scripts\activate"
  ✓ No need to remember "source venv/bin/activate"

  ✓ Automatic error checking (venv exists, .env exists)
  ✓ Clear error messages if something is missing

================================================================================
                          SIMPLE WORKFLOW
================================================================================

WINDOWS USERS:

1. Setup (one time):
   → Double-click: setup.bat

2. Configure (one time):
   → Edit .env file with your Jira credentials

3. Test connection:
   → Double-click: test_connection.bat

4. Run scan:
   → Double-click: run_scan.bat

5. View report:
   → Open reports\scan_XXXXXXXX_visual_analysis.html

---

LINUX/MAC USERS:

1. Setup (one time):
   → chmod +x *.sh
   → ./setup.sh

2. Configure (one time):
   → Edit .env file with your Jira credentials

3. Test connection:
   → ./test_connection.sh

4. Run scan:
   → ./run_scan.sh

5. View report:
   → Open reports/scan_XXXXXXXX_visual_analysis.html
